# Dashboard

- une fois authentifier l'utilisateur doit acceder a l'espace de formations et sur `dashboard` ou sûr il `learnings`
  doit voir

* Récent : Ce sont les cours qui sont recement mises en ligne (Ici quand on clique sur un cours, on tombe sur la page
  detail du cours)
* Découvrir : C'est tous les cours d'une manière générale (Ici quand on clique sur un cours, on tombe sur la page detail
  du cours)
* En cours : C'est le cours qui sont en cours d'apprentissage, c'est-à-dire que l'étudiant est en train d'apprendre. (
  Ici quand on tombe (on nous redirige) directement sur l'interface d'apprentissage)
* Terminé : Ce sont les cours que l'étudiant à deja finit d'apprendre (ici quand on clique sur le cours, on tombe sur le
  detail du cours avec des mentions cours terminer et si le cours est certifié ou pas, on doit signaler que le cours est
  certifié et avoir la possibilité de voir la certification)
* toujours sur `dashboard` lorsque sur `Reprenez votre apprentissage` c'est le cours qui est en cours d'apprentissage
  lorsque l'on clique sur l'action `Continuer`  on nous redirige directement sur la vue d'apprentissage

# En cours et Certifiantes

Sur l'URL `inprogress` on liste tous les cours que l'étudiant est en train d'apprendre avec progression inspiration des
cards existants sur `certificats` on liste tous les cours qui seront certifier à la fin de l'apprentissage
Un cours peut etre graduit ou payant

- pour un cours payant lorsque la personne accede a la page detail du cours et click sur demarrer le cours, il doit etre
  rediriger vers le paiement du cours avant de commencer a etudier une fois qu'il paie le cours, il est automatiquement
  redirger vers la page d'apprentissage
- pour un cours qui ne pas payant lorsqu'il clique sur demarrer le cours, il est automatiquement rediger vers la vue
  d'apprentissage
